# Game stat

# This application uses
-   [`Vue 3`](https://v3.ru.vuejs.org/ "`vue`")
-   [`GameStatus Public API`](https://gamestatus.info/ "`GameStatus Public API`")


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
